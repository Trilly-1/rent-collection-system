# Hero Background Image

Current image: `img1.jpg`

To change the background image:

1. Replace `img1.jpg` in this directory with your new image
2. Keep the same filename or update `src/pages/landing/LandingPage.jsx`
3. Recommended dimensions: 1920x1080px or larger
4. Format: JPG, PNG, or WebP

The hero section has a dark overlay gradient for text readability, so your image should work well with dark overlays.

If no image is found, the hero will use a dark navy background color as fallback.
